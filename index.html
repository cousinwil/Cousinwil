Author of Eeze

2012: Front-end development (LESS, CSS3, HTML5)

Eeze is a clean slate for professional developers creating custom applications. eeze.min.css comes out of the box at under 6K (BEFORE G-Zipping & you can gain ~2k by swapping to the HTML5 Reset) and sets up a great LESS workflow.

Unlike Twitter Bootstrap, Eeze comes as a minimal starting point for your app. I’ve only included the most common building blocks and base file structure that you might need to create a custom application. There are no base preset styles to work around, delete, or overwrite. Just clone the app and start writing *your* code in seconds.
