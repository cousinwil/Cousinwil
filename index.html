Author of Eeze (2012)

Front-end development (LESS, CSS3, HTML5)

Eeze is the logical conclusion of my older project, St.eeze. It uses HTML5 and LESS to deliver a clean slate from which to build a modern application front end. Unlike Twitter Bootstrap, Eeze comes as a clean slate. Included are the barest building blocks and base file structure one might need to create a custom application. There are no base colors or styles, no javascript includes. Just a great clean slate from which to start.
